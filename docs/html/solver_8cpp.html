Determines whether two webs interfere (i.e., overlap in live ranges).

Two webs interfere if they share at least one program line where both are considered active. End lines are not treated as active to avoid false conflicts when one web ends exactly where another starts.

Parameters
aFirst web.
bSecond web.
Returns
true if the webs interfere, false otherwise.
