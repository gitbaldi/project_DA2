Solver & solver )

Handles the command-line interface and program execution flow (T1.1).

  • This function validates the command-line arguments to ensure the program is running in batch mode ('-b'). It orchestrates the parsing of both input files (ranges and registers), sets up the solver, and triggers the register allocation algorithm and output generation.
  • Parameters
    argcThe number of command-line arguments.
    argvThe array of command-line argument strings.
    solverReference to the Solver object that manages the allocation process.
    Returns
    on success or -1 if any error occurs
  • Note
    Time Complexity: O(1) for argument validation. The overall complexity depends on the subsequent calls to parseRanges and parseRegisters.
